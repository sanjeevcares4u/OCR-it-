Problem Statement: OCR it!

Os: windows 10

Language Used: Python 3.6
Libraries used:
	opencv (Image Processing)
	Numpy (matrix manipulation)
	pytesseract (Image to text conversion)
	Pil (Text to image)

Installition :
Step 1. Python : Download python from python.org
Step 2. Set env.variable in path
Step 3. Using pip install opencv-python, numpy, pytesseract, PIl, base64 e.t.c
Step 4. Download tesseract from https://sourceforge.net/projects/tesseract-ocr-alt/files/tesseract-ocr-setup-3.02.02.exe/download.
Step 5. Install tesseract
Step 6. Set env. variable for tesseract as variable name: 'TESSDATA_PREFIX' and variable value: 'C:\Program Files (x86)\Tesseract-OCR'.
